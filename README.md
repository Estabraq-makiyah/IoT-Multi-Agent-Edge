Collaborative Decentralised Edge Computing with Multi-Agent Deep Q-Learning for Task Offloading

ABSTRACT: In the Internet of Things (IoT), sensors, gateways, and services exist at multiple levels of interaction that work at different levels to provide the end user with a complete service. Different domains deploy IoT networks for specific tasks that can be monitored remotely. The increase in connected IoT devices and their notable, limited computational power calls for resource-efficient and in-between layers of task processing on the network. This paper presents an innovative framework that employs a multi agent deep Q learning framework as a response to challenges pertaining to resource adequacy in decentralised edge computing. This utilizes a multi-agent DRL approach in such a way that every edge node behaves like an individual agent of its kind, learning optimal decisions relating to task offloading as well as resource allocation. Here, decisions will be made based on the local observations found in the environment and hence no central controller is required; thus, lowering communication overheads. The agents are trained using a deep Q-learning algorithm through interaction with the environment for finding optimal policies by means of resource consumption and delay under certain constraints. Simulations provide proof that proposed methodologies are the ones that improve substantially resource utilization together with reducing the time of completion more than any conventional methods.
